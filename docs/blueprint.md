# **App Name**: Bana'i Tracker

## Core Features:

- Expense Recording: Record expenses related to a specific project or line item within a project.
- Supplier Accounts: Manage supplier accounts, including transactions and balances.
- Customer Accounts: Manage customer accounts, tracking invoices and payments.
- Employee Management: Maintain employee records, track advances, and manage expense settlements.
- Reporting: Generate supplier reports.
- Insight Generation: Use generative AI tool to provide insights.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Teal (#009688) for interactive elements and important notifications.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable user experience.
- Use simple, outlined icons from a library like Material Icons for a clean and consistent look.
- Maintain a clean and organized layout, using a grid system to ensure consistency across all pages.
- Subtle animations on data load. Consider subtle transitions to show project expense data as they load in.
